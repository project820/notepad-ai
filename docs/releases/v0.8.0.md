# v0.8.0

## 새로운 기능

### HTML 응답을 문서로 더 정확히 추출합니다

성공적으로 처리한 HTML 응답에서는 위험 요소만 자동으로 정리해 문서로 만들고, 응답이 비어 있거나 해석할 수 없으면 명확한 오류를 표시합니다. 코드 펜스나 설명문으로 감싼 응답에서도 문서를 추출하며, 조각 응답에서는 주변 설명문을 제거합니다.

### 바로 시작하는 위저드

HTML 생성 탭을 열면 위저드가 바로 표시됩니다. 빠르게 만들 때는 자동 흐름을, 더 직접 정하고 싶을 때는 디테일 흐름을 선택해 용도·표현 스타일·방향·밀도를 정할 수 있습니다.

### 모델 선택을 유지한 재시도

생성이 끝나지 않아도 고른 모델은 그대로 유지됩니다. 다시 시도해도 같은 설정에서 이어갈 수 있고, 생각 시간이 긴 모델은 첫 응답을 최대 240초까지 기다립니다.

### 문제를 더 쉽게 확인하는 진단 로그

앱 로그를 파일로 남기고 최근 3일치를 보관합니다. HTML 생성 중 정리되거나 실패한 항목의 코드를 함께 기록해, 문제가 생겼을 때 원인을 더 쉽게 확인할 수 있습니다.

### Grok Composer 연결을 명확하게 안내합니다

Grok Composer는 xAI API 키가 있을 때만 선택할 수 있습니다. 설정에 xAI API 키 입력란을 추가했으며, 키가 없으면 설정 방법을 현지화된 오류로 안내합니다.

### HTML 내보내기 모델과 Fast 옵션을 정리했습니다

HTML 내보내기에는 검증된 ChatGPT(GPT-5.6 Sol·Terra·Luna), Claude, Grok 및 자동 감지한 로컬 모델만 표시합니다. GPT-5.6 모델에서는 Fast 옵션을 선택해 더 빠른 응답을 요청할 수 있습니다.

## 함께 개선된 점

- 화면용 CSS와 기본 SVG 네임스페이스를 포함한 문서 표현을 더 자연스럽게 유지합니다.
- 한 번의 생성에서도 글자 크기·밀도·링크·로컬 모델 선택을 더 안정적으로 반영합니다.
