# v0.9.0

## 인터랙티브 HTML 내보내기

### 밝고 어두운 테마를 바로 전환합니다

내보낸 HTML 문서 오른쪽 위에서 ☀/🌙 버튼으로 밝은 테마와 어두운 테마를 바꿀 수 있습니다. 선택한 테마는 다음에 문서를 열어도 유지됩니다.

### 슬라이드를 더 편하게 넘깁니다

슬라이드 형식으로 내보내면 ←/→ 키와 PageUp/PageDown, Space 키로 넘길 수 있습니다. 화면의 이전/다음 버튼이나 진행 표시를 클릭해도 이동하며, 인쇄할 때는 모든 슬라이드가 이어서 표시됩니다.

### 문서 안에서 직접 반응하는 요소를 만듭니다

생성된 HTML에서는 체크박스를 선택하는 것처럼 폼 컨트롤과 모델이 만든 상호작용을 사용할 수 있습니다. 탭, 아코디언, 간단한 애니메이션처럼 문서 안에서 완결되는 기능을 담을 수 있습니다.

### 네트워크 연결 없이 실행됩니다

내보낸 문서는 필요한 코드와 스타일을 문서 안에 담아 실행합니다. 콘텐츠 보안 정책(CSP)이 네트워크 요청과 폼 전송을 차단하므로, 인터랙티브 요소도 외부 서비스에 연결하지 않습니다.
