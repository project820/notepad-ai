Step 4 / 5

확인 창에서 "열기" 클릭

우클릭 → 열기로 실행하면 macOS가 한 번 더 확인을 요청합니다. 열기 버튼을 클릭하세요.

!
"Notepad AI" 앱을 열까요?
macOS는 이 앱의 개발자를 확인할 수 없어 악성 소프트웨어인지 확인할 수 없습니다. 신뢰할 수 있는 곳에서 받은 경우에만 여세요.
✅ 이건 최초 1회만 입니다. 이후 더블클릭으로 평소처럼 실행돼요.

방법 B — 시스템 설정에서 허용

실행이 거부된 직후 곧바로:

  1. 시스템 설정 → 개인정보 보호 및 보안 열기
  2. 아래쪽 "보안" 섹션에서 "Notepad AI이(가) 차단되었습니다" 메시지 옆 그래도 열기 클릭
  3. 로그인 비밀번호 입력

방법 C — 터미널 한 줄

기술에 익숙하다면 ⌘+스페이스로 터미널을 열고 한 줄 실행:

xattr -dr com.apple.quarantine "/Applications/Notepad AI.app"

실행 후 평소처럼 더블클릭하면 열립니다.

이 명령은 macOS의 "이 파일은 인터넷에서 받음" 표시를 제거합니다. 직접 빌드했거나 공식 릴리스에서 받은, 신뢰할 수 있는 파일에만 사용하세요.

AI 기능 연결 (선택)

편집기는 AI를 연결하지 않아도 완전히 사용할 수 있습니다. AI 버튼에서 사용할 연결 방식을 고르고 안내에 따라 설정하세요.

구독 또는 API 연결
ChatGPT 구독 로그인, Claude·Grok 로컬 CLI, Anthropic·OpenRouter API 키를 지원합니다.
로컬 모델
Ollama 또는 LM Studio를 실행해 두면 설치된 모델을 찾아 연결할 수 있습니다.
API 키와 로그인 정보는 macOS Keychain으로 암호화해 저장합니다. 연결 방식과 모델은 설정에서 언제든 바꿀 수 있습니다.

문서 가져오기

기존 문서를 마크다운으로 변환해 편집할 수 있습니다. 메뉴 파일 → 열기… 또는 ⌘ + O로 파일을 선택하세요.

DRM이 걸린 문서나 이미지뿐인 스캔 PDF는 변환되지 않거나 결과가 제한될 수 있습니다.

AI 기능 소개

연결한 모델을 문서 작업에 맞춰 선택해 사용할 수 있습니다.

통합 채팅
작성, 상담, 프로젝트 설정, HTML 생성을 한 패널에서 사용합니다.
블록 AI
선택한 문장을 다듬거나 대안을 받아 문서에 적용합니다.
이미지와 OCR
이미지를 첨부하면 비전 모델을 사용하거나 기기 내 OCR로 텍스트를 읽습니다.
HTML 내보내기
문서를 스타일이 적용된 독립형 HTML 페이지나 슬라이드로 만듭니다.
모델은 기능별로 따로 선택할 수 있으며, 로컬 모델을 쓰면 문서를 외부 서비스로 보내지 않아도 됩니다.

🎉 준비 완료 — 첫 사용 가이드

5초 시작

  • 아무 영역에 클릭하고 그냥 글 작성
  • 중요한 단어 드래그 → 툴바의 B 클릭 → 굵게
  • 새 줄에 H1 클릭 → 큰 제목
  • ⊞ Table 클릭 → 격자에서 표 크기 선택 → 클릭
  • 오른쪽 리치뷰에서 직접 워드처럼 편집해도 OK

자주 쓰는 단축키

⌘ N / ⌘ O새 문서 / 문서 열기 ⌘ S / ⇧⌘ S저장 / 다른 이름으로 저장 ⌘ P프리뷰 보기 전환

개인 설정

우상단 메뉴에서 언어(한국어/English), 다크 모드, 글꼴 크기를 변경할 수 있습니다. 설정은 자동으로 저장됩니다.

💡 상단의 제목 영역(Untitled)을 클릭해 파일명을 먼저 지정하고 작성을 시작할 수도 있습니다.

문제가 생기면 GitHub 저장소의 Issues에 남겨주세요.
