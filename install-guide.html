처음 실행 — Gatekeeper 우회

Notepad AI는 오픈소스 프로젝트라 Apple 개발자 서명 비용을 들이지 않았습니다. macOS가 "확인되지 않은 개발자" 경고를 띄울 텐데, 한 번만 우회하면 됩니다.

방법 A — 우클릭 → 열기 (권장)

1. Applications 폴더에서 Notepad AI 아이콘을 Control + 클릭 (또는 우클릭)
2. 메뉴에서 "열기" 선택.

Step 4 / 5

확인 창에서 "열기" 클릭

우클릭 → 열기로 실행하면 macOS가 한 번 더 확인을 요청합니다. 열기 버튼을 클릭하세요.

!
"Notepad AI" 앱을 열까요?
macOS는 이 앱의 개발자를 확인할 수 없어 악성 소프트웨어인지 확인할 수 없습니다. 신뢰할 수 있는 곳에서 받은 경우에만 여세요.
✅ 이건 최초 1회만 입니다. 이후 더블클릭으로 평소처럼 실행돼요.
Step 5 / 5 (옵션)

그래도 안 열리면?

macOS 15(Sequoia)부터는 "열기" 옵션이 안 보일 수 있습니다. 두 가지 대안:

방법 B — 시스템 설정에서 허용

실행이 거부된 직후 곧바로:

  1. 시스템 설정 → 개인정보 보호 및 보안 열기
  2. 아래쪽 "보안" 섹션에서 "Notepad AI이(가) 차단되었습니다" 메시지 옆 그래도 열기 클릭
  3. 로그인 비밀번호 입력

방법 C — 터미널 한 줄

기술에 익숙하다면 ⌘+스페이스로 터미널을 열고 한 줄 실행:

xattr -dr com.apple.quarantine "/Applications/Notepad AI.app"

실행 후 평소처럼 더블클릭하면 열립니다.

이 명령은 macOS의 "이 파일은 인터넷에서 받음" 표시를 제거합니다. 직접 빌드했거나 공식 릴리스에서 받은, 신뢰할 수 있는 파일에만 사용하세요.

AI 기능 활성화 — ChatGPT 로그인

앱이 처음 실행되면 AI 로그인 창이 자동으로 나타납니다. 개인 ChatGPT Plus 계정을 사용합니다.

AI 기능 없이도 편집기는 완전히 동작합니다. 로그인은 나중에 F7로 다시 열 수 있습니다.
1
"로그인" 버튼 클릭 → 브라우저가 auth.openai.com으로 이동
2
앱에 표시되는 코드를 브라우저에 입력
ABCD-1234
3
ChatGPT 계정으로 로그인 → 완료
✅ 로그인 정보는 macOS Keychain에 안전하게 저장됩니다. 재실행해도 자동 로그인.

AI 기능 소개

로그인 후 네 가지 AI 도우미를 사용할 수 있습니다.

⌘⇧A 또는 텍스트 선택 → ✦ 버튼
블록 AI — 문장 수정
선택한 텍스트를 3가지 대안으로 다시 씁니다. 마음에 드는 것을 "적용"하면 즉시 교체.
⌘J
사이드 채팅 — 문서 상담
오른쪽 패널에서 AI와 질문/답변. 문서를 직접 수정하지 않고 조언만 받을 때.
⌘ ;
하단 채팅 — 초안 작성
화면 아래 채팅창에서 초안 요청. "적용" 버튼으로 에디터에 바로 삽입.
F6
글쓰기 수준 조절
초등~전문가 5단계. 설정하면 모든 AI 답변의 문체가 해당 수준에 맞춰집니다.
💡 툴바 오른쪽 끝 ✦ 모델 버튼을 클릭하면 사용할 AI 모델을 선택할 수 있습니다.

🎉 준비 완료 — 첫 사용 가이드

5초 시작

  • 아무 영역에 클릭하고 그냥 글 작성
  • 중요한 단어 드래그 → 툴바의 B 클릭 → 굵게
  • 새 줄에 H1 클릭 → 큰 제목
  • ⊞ Table 클릭 → 격자에서 표 크기 선택 → 클릭
  • 오른쪽 리치뷰에서 직접 워드처럼 편집해도 OK
