## 7\. 설비 소싱 — 리서치 결론 {#lp-h-13}

### 7-1. Hard dough cookie (차갑게 압출 → 치밀 조직) {#lp-h-14}

-   조직감 보존력: **로터리 몰더 > 압출+와이어컷 > 와이어컷 디포지터**
-   **순수 치밀식감 최우선 → 로터리 몰더**(Polin RTV / MIMAC, 유럽사·수입원 경유)
-   **인클루전(초코칩·견과) 포함 → 와이어컷**(Masdac H1) 필수
-   칠링 도우: 압출식은 **호퍼 냉각기능** 모델로. 로터리는 온도영향 적음
-   FOOMA 현장: **Rheon 중첩노즐 와이어컷 슬라이서** = 동결 없이 하드도우 컷(효율↑)

### 7-2. Soft dough cookie (츠마가리·Snaffles 류) {#lp-h-15}

-   부드러운 드롭쿠키 → **펌프/피스톤 디포지터**(젠틀 토출, 테일링 방지). 소배합·다품종이면 **MIMAC Babydrop**(헤드 3종 교체) 가성비 우수(이탈리아사)
-   필링/샌드 소프트쿠키 → **Rheon 인크러스팅(KN/CN)** 골드스탠다드
-   검증: 반죽 샘플 지참 라이브 데모, **히티드 호퍼** 여부 확인

### 7-3. 2구 호퍼 (2종 도우 혼합 출력 OR 포앙) — ⭐핵심 {#lp-h-16}

-   “2종 도우 혼합 출력” = **이색/코익스트루전** (Mono 트리플호퍼, Tonelli — FOOMA 미출전)
-   “포앙” = **인크러스팅 = Rheon 독점적 1위**
-   **결론: Rheon 하나로 두 기능 커버** — 신형 **CN700 + 디포지터 옵션**(포앙+기초 이색) 또는 **Multi Co-Extruder**(전용 이색라인). 단일 생태계 + 일본 A/S → 1순위 부스 W2-36-24

### 7-4. 배합실 믹싱 (일 강력분 500kg~1.5t) {#lp-h-17}

-   **대원칙: 빵용=스파이럴 / 치아바타=포크 / 제과=플래니터리 — 분리 필수** (스파이럴·포크는 크림·머랭 불가)
-   **이탈볼(removable bowl)** = 다품종 소량의 핵심(볼 교체로 10분 내 다음 반죽)

| 용도 | 믹서 | 권장 반죽용량 | 모델 예시 |
| --- | --- | --- | --- |
| 식빵+바게트(대형) | 스파이럴, 이탈볼 | 120~250kg | WP-Kemper SP160~250 (대아상교 컨택) |
| 치아바타(고가수율) | **포크 믹서** | 60~80kg | Mecnosud FC / Sancassiano |
| 소형 빵/특수 | 스파이럴, 이탈볼 | 45~80kg | Kemper SP45~80 / Esmach |
| 제과·쿠키 | 플래니터리 | 10~40kg | Aicohsha ACM / Kanto 소형 |

-   1.5t/일: 대형 스파이럴1 + 중형 스파이럴1 + 포크1 + 플래니터리1 구성 권장
-   FOOMA 국산 즉시구매: Kanto #451 · Aicohsha #413 · 関西久保長 #1033

## 8\. 일본/유럽 vs 중국 소싱 — 중국 구매리스트 검증 {#lp-h-18}

**원칙: 핵심 정밀공정 = 일본/유럽, 보조설비 = 중국.** 절감분을 Rheon 성형기·정밀 믹서/분할기에 투자.

| 첨부 중국리스트 항목 | 판정 | 비고 |
| --- | --- | --- |
| Automatic panning(자동 팬닝) | 🟢 중국 OK | 기계식 |
| 믹싱볼·빵박스·철판 세척 | 🟢 중국 OK | 범용 고압세척 |
| Dough 청커 & 리프터 벨트 | 🟢 중국 OK | 정밀도 불필요 |
| **Bag dump station (1.5t)** | 🟢 중국 OK | SINOBAKE 등, 가성비 큼 |
| 제함기/디패닝(Suun true류) | 🟢 중국 OK | 초음파 디패닝 |
| Roll cake 자동화 | 🟡 분리 | 롤링·스프레이=중국 / **크림 디포지터=일본·유럽(±1g)** |
| 케이크 시트 아이싱 자동화 | 🟡 분리 | 이송=중국 / **아이싱 헤드=유럽(Unifiller)** |
| 충전기(슈크림/잼/소스, 로보쿠보 대체) | 🟡 분리 | 단순=중국 / **정밀 포션=일본·유럽** |
| 치아바타 공정 자동화 | 🟡 분리 | 발효·펀칭·팬닝=중국 / **분할기(divider)=일본**(중량 ±1~2% vs 중국 ±5~10%) |
| **Rheon 포앙기 본체** | 🔴 일본(Rheon) | 보조만 중국, 포앙기 자체 대체 금지 |
| 빵 반죽 믹서 | 🔴 일본/유럽 | 온도제어·글루텐 품질 |

### 중국 설비 구매 시 필수 체크 {#lp-h-19}

-   **KC인증 + 식품위생법**(SUS304/316, FDA 실링) 문서 요청
-   **예비부품 2년치** 협상(한국 A/S망 부재), 수명 5~7년(일본 15~20년) 감안
-   IoT 연계 필요 시 제어반 **지멘스 S7-1500 / 미쓰비시 iQ-R 업그레이드** 옵션

### 권장 소싱 비율(예산 기준) {#lp-h-20}

-   일본 ~50~55% (Rheon 성형, 믹서, 치아바타 분할기)
-   유럽 ~10~15% (정밀 디포지터, 프리미엄 제과)
-   중국 ~30~35% (팬닝·세척·이송·백덤프·디패닝·롤링)
